# 🎮 PixelZone

<p align="center">
  PixelZone es la plataforma de juegos cognitivos y de habilidad diseñada para desafiar tu mente.<br/>
  Registrate, iniciá sesión y competí en Ahorcado, Mayor o Menor, Preguntados y más.<br/>
  Seguí tu progreso con estadísticas detalladas, chateá con otros jugadores en tiempo real<br/>
  y trepá al top del ranking. ¡Demostrá quién tiene los mejores reflejos y la mente más ágil!
</p>
